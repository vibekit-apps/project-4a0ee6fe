# VibeKit Hello World

A beautiful hello world page built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy

Deploy easily to Vercel:

```bash
npm run build
```

The app is ready for production deployment on Vercel or any other hosting platform.
